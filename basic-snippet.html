The Snippet

The snippet contains two different tags. The first being the paragraph tag and the next being the hyperlink tag.

Relationship Between Tags:

The hyperlink tag is a child of the paragraph tag.

Attribute Names and Values of the Hyperlink Tag:

  • href: https://google.com
  • id: the-best-link
  • class: banana
  • target: _blank

Purpose and Effect of Above Attributes:

The "href" attribute gives the URL of the page the link goes to

The "id" attribute gives a unique id for an element

The "class" attribute refers to a class in the stylesheet

The "target" attribute indicates where the linked element will open. In this case it will open in a new tab.
